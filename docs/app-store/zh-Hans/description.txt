Mimi Remote 是一款原生移动开发工作台，用来连接你拥有或获授权使用的计算机开发环境。

连接后，你可以创建或继续编码会话、查看代码修改、确认受限操作，并在所有命令仍由你控制的计算机执行的前提下继续开发工作。

主要功能：
- 浏览、搜索、创建和继续编码会话
- 查看结构化回复、推理过程、代码块、图片和 Diff
- 在受限开发操作运行前进行确认
- 查看 Git 状态并管理分支与 worktree
- 使用文字、图片和设备端语音输入
- 为不同计算机保存相互隔离的连接档案
- 通过局域网、私有网络或 HTTPS 连接

Mimi Remote 只与用户配置的主机直接通信。App 不下载或在移动设备上执行代码，也不提供模型、第三方账号、订阅、云端代码执行、VPN、流量中转或项目托管。项目内容和连接凭据不会发送给 Mimi Remote 开发者。

使用要求：
- 一台已安装 Mimi 配套服务和兼容开发运行时的计算机
- 移动设备能够通过网络访问所配置的计算机
- 只连接你拥有或已获明确授权访问的设备与项目

Mimi Remote 是独立开发的开源工具，不隶属于任何兼容工具提供方，也不代表其官方产品。
