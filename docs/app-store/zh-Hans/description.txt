Mimi Remote 是一款原生 iPhone 和 iPad 客户端，用来连接你拥有或获授权使用的 Mac 上的 Codex。

连接 Mac 后，你可以创建或继续编码会话、查看代码修改、审批待执行操作，并在代码仍由 Mac 执行的前提下继续开发工作。

主要功能：
- 浏览、搜索、创建和继续 Codex 会话
- 查看结构化回复、推理过程、代码块、图片和 Diff
- 在受限开发操作运行前进行审批
- 查看 Git 状态并管理分支与 worktree
- 使用文字、图片和可选语音输入
- 为你控制的不同 Mac 保存相互隔离的连接档案
- 通过局域网、Tailscale 等私有网络或 HTTPS 连接

Mimi Remote 不提供 AI 模型、AI 订阅、云端代码执行、VPN 或项目托管。你需要在自己的 Mac 上安装并运行 Codex 和 Mimi 配套服务。模型提供方通信由该 Mac 上配置的工具发起，不经过 Mimi Remote 开发者运营的服务。

用户还可以在 Mac 上单独启用实验性的 Claude Code bridge。它不是 Codex 核心体验的必要条件，Mimi Remote 也不提供 Claude 账号或订阅。

使用要求：
- 一台已经安装并配置 Codex CLI 的兼容 Mac
- Mac 上已经安装 Mimi 配套服务
- iPhone 或 iPad 能够通过网络访问所配置的 Mac

Mimi Remote 是独立开发的开源第三方客户端，不隶属于 OpenAI、Anthropic、Tailscale 或 GitHub，也不代表这些平台的官方产品。
